# Supply Chain Performance Dashboard

A comprehensive interactive dashboard for monitoring and analyzing supply chain metrics, sales trends, and shipping performance.

[Supply Chain Dashboard](https://ibb.co/b5LCh79m)

## Overview

This dashboard provides real-time insights into supply chain operations, tracking key performance indicators (KPIs) across revenue, orders, delivery times, and shipping modes.

## Key Features

### Performance Metrics
- **Total Revenue**: $36.78M tracked across all operations
- **Total Orders**: 66K orders processed
- **Average Delivery Days**: 3.5 days
- **On-Time Delivery Rate**: 17.83%

### Visualizations

**Sales Trend Analysis**
- Time series visualization from 2015-2018
- Identifies seasonal patterns and growth trends
- Notable spike in late 2017

**Sales by Category**
- Top performing categories: Fishing, Cleats, Cardio Equipment
- Comprehensive breakdown across 15+ product categories
- Visual comparison of category performance

**Shipping Performance Table**
- Four shipping modes analyzed: First Class, Same Day, Second Class, Standard Class
- Metrics include total orders, average delivery days, on-time delivery %, and late deliveries
- Same Day shipping shows highest on-time delivery rate (49.26%)

**Geographical Sales Map**
- Global distribution of sales across continents
- Concentration in North America, Europe, and Asia
- Interactive visualization for regional analysis

### Interactive Filters
- Category Name
- Shipping Mode
- Market
- Department Name
- Order Status
- Customer Segment

## Insights

- Same Day shipping achieves the best on-time delivery performance at 49.26%, despite having fewer total orders
- Standard Class handles the highest volume (39,324 orders) but maintains consistent delivery times
- Sales show strong growth through 2017 with a significant spike in Q4
- Fishing and Cleats categories dominate sales performance
